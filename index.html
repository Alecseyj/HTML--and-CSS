Что такое CSS и HTML в краце ?

Что такое CSS ?

CSS — это формальный язык описания внешнего вида документа, написанного с использованием языка разметки. Также может применяться к любым XML-документам, например, к SVG или HTML. (Внешний вид сайта и т.п.)

Что такое HTML ?

HTML — стандартизированный язык гипертекстовой разметки документов для просмотра веб-страниц в браузере. Веб-браузеры получают HTML документ от сервера по протоколам HTTP/HTTPS или открывают с локального диска, далее интерпретируют код в интерфейс, который будет отображаться на экране монитора. (Базовая структура сайта)
